# Introduction: Ethereum from 50,000 feet

Ethereum is a verifiable, permissionless, and permanent computing platform.

- Verfiable: you can verify the output of your input to Ethereum.
- Permissionless: anyone with an internet connection can use Ethereum.
- Permanent: Ethereum is designed to last forever.  When you deploy an application on Ethereum, unless you design it such that it can be shut down, it will last forever.

From the vintage point of 2026 (when this sentence was written), it has become clear that *money* and *finance* in general can benefit greatly from these properties, since:

- Money needs to be veriable.
- Money is more useful when anyone can use it.
- Money needs to last forever.

However, there are other types of applications that are starting to find product-market fit on Ethereum, such as prediction markets.

## Ethereum vs AWS

AWS is not veriable -- users who use Twitter have no way to now how Twitter processes and recommends its tweets.

AWS is not permissionless -- not everyone is allowed to use it.

AWS is not decentralized -- if Amazon shuts down, AWS is gone.

## What applications benefit from these properties?

- Finance
  - Money
    - Store of value (ETH)
    - Medium of exchange (stablecoins)
  - Exchanges
    - Spot
    - Perpetuals

- Information markets
  - Prediction markets
