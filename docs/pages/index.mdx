# Introduction: Ethereum from 50,000 feet

Ethereum is a verifiable, permissionless, and decentralized computer.

- Verfiable: you can verify the output of your input to Ethereum.
- Permissionless: anyone with an internet connection can use Ethereum.
- Decentralized: the computer is powered by a network of nodes across the globe, and so long as a large enough portion of the nodes are running, the computer will continue to function.

Let's zoom out and compare Ethereum with other computing platforms in order to understand why these two properties make Ethereum unique.

## Ethereum vs AWS

AWS is not veriable -- users who use Twitter have no way to now how Twitter processes and recommends its tweets.

AWS is not permissionless -- not everyone is allowed to use it.

AWS is not decentralized -- if Amazon shuts down, AWS is gone.

## What applications benefit from these properties?

  - Money

  - Exchanges

- Information markets
  - Prediction markets
